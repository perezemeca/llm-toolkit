# Caveman-Codex

Usar Caveman solo para reportes de programación con Codex. No reemplaza RTK: RTK compacta salidas de comandos y Caveman compacta respuestas del agente.

## Niveles

- lite: respuesta breve, gramática normal, sin relleno.
- full: fragmentos, menos artículos, mayor compresión.
- ultra: telegráfico, solo para reportes mecánicos.

## Preservar siempre

- rutas de archivos;
- comandos ejecutados;
- errores exactos;
- nombres de tests;
- resultados de tests;
- riesgos técnicos.

## Alcance

Caveman aplica únicamente al modo compacto de programación con Codex.
